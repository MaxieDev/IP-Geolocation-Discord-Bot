### Installation:
- OS Ubuntu 20.04
- Python Version 3.9
```
apt-get update
apt-get install git
git clone https://github.com/
apt-get install python3.9
apt-get install python3-pip
pip3 install discord.py
pip3 install requests
nohup python3 Area51.py &
```
